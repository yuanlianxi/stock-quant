## 1. 项目概述

### 1.1 目标
搭建一个可复用的量化回测平台，支持数据获取→策略回测→绩效评估→报告生成的完整流程。

### 1.2 核心价值
- 降低策略验证成本（回测 → 再实盘）
- 支持多策略对比
- 输出标准化的绩效报告

## 4. 模块设计

### 4.1 数据层（DataLoader）

**职责**：统一的数据获取接口，支持多数据源

**接口**：
```python
class DataLoader:
    def get_stock_daily(self, code: str, start: str, end: str) -> pd.DataFrame
    def get_etf_daily(self, code: str, start: str, end: str) -> pd.DataFrame
    def save_to_sqlite(self, df: pd.DataFrame, table: str)
    def load_from_sqlite(self, table: str, code: str) -> pd.DataFrame
```

**数据源**：
| 数据源 | 用途 | 优先级 |
|--------|------|--------|
| akshare | 股票、ETF、期货 | P0 |
| baostock | 股票复权数据 | P1 |

**存储**：SQLite（`data/stock_quant.db`）

---

### 4.2 回测引擎（BacktestEngine）

**职责**：封装 Backtrader，提供统一的回测接口

**接口**：
```python
class BacktestEngine:
    def __init__(self, config: dict)
    def add_data(self, datafeed)
    def add_strategy(self, strategy_class, **params)
    def run(self) -> Cerebro
    def get_results(self) -> dict
```

**配置参数**：
| 参数 | 默认值 | 说明 |
|------|--------|------|
| initial_cash | 1000000 | 初始资金 |
| commission | 0.0003 | 手续费（万三）|
| stamp_tax | 0.001 | 印花税（千一）|
| slippage | 0.0001 | 滑点 |

---

### 4.3 策略基类（StrategyBase）

**职责**：定义策略开发规范，预置公共方法

**接口**：
```python
class StrategyBase(bt.Strategy):
    def __init__(self):
        self.order = None
        self.buy_price = None
        self.buy_comm = None

    def notify_order(self, order)
    def notify_trade(self, trade)
    def get_metrics(self) -> dict
```

---

### 4.4 绩效评估（Performance）

**职责**：计算策略绩效指标

**指标**：
| 指标 | 计算方式 |
|------|---------|
| 总收益率 | (期末净值 - 1) × 100% |
| 年化收益率 | 总收益率 / 回测年数 × 100% |
| 夏普比率 | (年化收益 - 无风险利率) / 年化波动率 |
| 最大回撤 | max(peak - current) / peak |
| 胜率 | 盈利交易数 / 总交易数 |
| 盈亏比 | 平均盈利 / 平均亏损 |

---

### 4.5 报告生成（Report）

**职责**：生成 HTML 回测报告

**输出内容**：
- 绩效指标汇总表
- 收益曲线图
- 回撤曲线图
- 持仓变化图
- 交易记录表

## 5. 预置策略

### 5.1 双均线策略（DualMA）

**参数**：
| 参数 | 默认值 | 说明 |
|------|--------|------|
| fast_ma | 5 | 快线周期 |
| slow_ma | 20 | 慢线周期 |

**规则**：
- 金叉（快线上穿慢线）→ 买入
- 死叉（快线下穿慢线）→ 卖出

---

### 5.2 海龟策略（Turtle）

**参数**：
| 参数 | 默认值 | 说明 |
|------|--------|------|
| entry_period | 20 | 入场通道周期 |
| exit_period | 10 | 出场通道周期 |
| atr_period | 20 | ATR周期 |
| risk_unit | 0.02 | 单笔风险比例 |

**规则**：详见 `notebooks/research/0005_海龟交易策略深度调研.md`

---

### 5.3 网格策略（Grid）

**参数**：
| 参数 | 默认值 | 说明 |
|------|--------|------|
| grid_num | 10 | 网格数量 |
| grid_ratio | 0.02 | 网格间距比例 |

**规则**：
- 价格每上涨一格 → 卖出
- 价格每下跌一格 → 买入

## 6. 使用流程

### 6.1 数据准备
```bash
# 首次运行自动获取数据
python scripts/run_backtest.py --codes 510300 --start 2018-01-01 --end 2023-12-31
```

### 6.2 运行回测
```bash
# 双均线策略
python scripts/run_backtest.py --strategy dual_ma --codes 510300 --start 2018-01-01 --end 2023-12-31

# 海龟策略
python scripts/run_backtest.py --strategy turtle --codes 510300 --start 2018-01-01 --end 2023-12-31

# 多策略对比
python scripts/run_backtest.py --strategy all --codes 510300 --start 2018-01-01 --end 2023-12-31
```

### 6.3 查看报告
```bash
# 报告生成在 output/ 目录
open output/report_510300_2028-04-28.html
```

## 9. 风险与限制

1. **数据质量**：akshare 免费数据可能存在缺失，需定期校验
2. **滑点**：实盘滑点难以精确模拟，报告中滑点参数仅供参考
3. **过拟合**：回测结果不代表实盘表现，需小资金验证
4. **未包含**：涨停、跌停、T+1 等 A股特有交易规则
