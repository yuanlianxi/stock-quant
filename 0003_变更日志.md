# Stock Quant - 变更日志

## [v0.5.1] - 2026-04-28

### 新增
- `backtest/` 目录：Phase 1 回测系统数据层实现
  - `backtest/__init__.py`：模块初始化
  - `backtest/config.py`：配置（数据路径、数据库路径、默认参数、数据源配置）
  - `backtest/data_loader.py`：DataLoader 类，统一数据获取接口，支持 SQLite 本地缓存 + akshare 在线获取
- `scripts/run_backtest.py`：回测运行样例脚本
- `requirements.txt`：追加 akshare、baostock 依赖

### 功能
- `DataLoader.get_stock_daily()`：获取股票日线数据（支持 SQLite 缓存 + akshare 获取）
- `DataLoader.get_etf_daily()`：获取 ETF 日线数据（支持 SQLite 缓存 + akshare 获取）
- `DataLoader._init_db()`：自动创建 stock_daily、etf_daily 表
- `get_data_loader()`：便捷函数，返回 DataLoader 单例

### 状态
- Phase 1（数据层）：✅ 已完成
