# Stock Quant - 项目设计

## 1. 项目概述

- **项目名称**：Stock Quant
- **项目类型**：股票量化研究与工具开发
- **目标**：调研股票量化分析方法，开发量化交易工具

## 2. 项目阶段

### 阶段一：调研（当前阶段）
- 学习量化交易基础知识
- 了解主流量化框架和数据源
- 熟悉 Python 量化工具链
- 输出调研报告

### 阶段二：工具开发（待定）
- 根据调研结果确定技术方案
- 开发回测系统/实盘工具

## 3. 技术栈

| 工具 | 用途 |
|------|------|
| Python | 主语言 |
| pandas/numpy | 数据分析 |
| akshare | 财经数据获取 |
| backtrader | 回测框架（可选）|
| jupyter | 调研分析 |
