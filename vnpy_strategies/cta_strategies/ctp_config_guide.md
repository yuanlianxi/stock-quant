# CTP 连接配置指南

本文档说明 VN.py 连接 SimNow 仿真环境及实盘期货公司 CTP 的配置方法。

---

## 1. SimNow 仿真环境

SimNow 是 CTP 官方提供的期货仿真交易平台，用于策略回测验证和模拟交易。

### 基础信息

| 项目 | 值 |
|------|-----|
| Broker ID | 9999 |
| 交易服务器 | `180.168.146.187:10101` |
| 行情服务器 | `180.168.146.187:10111` |
| 认证服务器 | `180.168.146.187:10130` |
| 用户名 | 任意 11 位数字（手机号格式） |
| 密码 | 任意 6 位以上字符 |

### VN.py 连接配置示例

在 VN Station 或 `run.py` 中配置：

```python
# SimNow 连接参数
ctp_setting = {
    "用户名": "your_mobile_number",
    "密码": "your_password",
    "经纪商代码": "9999",
    "交易服务器": "180.168.146.187:10101",
    "行情服务器": "180.168.146.187:10111",
}
```

> ⚠️ SimNow 行情和交易使用**同一个连接**，部分券商实盘可能分开。

## 2. 实盘期货公司连接

### 2.1 获取连接参数

联系您的期货公司客户经理，获取以下信息：

| 参数 | 说明 | 示例 |
|------|------|------|
| Broker ID | 期货公司代码 | 0081（华泰）|
| 交易服务器 | Trade Frontend | `121.37.81.82:41205` |
| 行情服务器 | Market Frontend | `121.37.81.82:41213` |
| 用户名 | 期货账户 | 8位数字 |
| 交易密码 | 账户密码 | - |
| 动态密钥 | 软Token/手机验证码 | - |

### 2.2 国内主要期货公司 Broker ID 参考

| 期货公司 | Broker ID |
|----------|-----------|
| 华泰期货 | 0081 |
| 中信期货 | 9999（SimNow）/ 各公司独立 |
| 国泰君安 | 0012 |
| 银河期货 | 0011 |
| 方正中期 | 0255 |
| 南华期货 | 0099 |

> ⚠️ Broker ID 以期货公司官方提供为准，上表仅供参考。

### 2.3 实盘配置示例

```python
# 以华泰期货为例
realtime_setting = {
    "用户名": "88888888",       # 期货账号（8位）
    "密码": "your_password",
    "经纪商代码": "0081",
    "交易服务器": "121.37.81.82:41205",
    "行情服务器": "121.37.81.82:41213",
}
```

## 3. VN.py CTPSTD 接口（推荐）

对于支持新一代 CTP STD 接口的期货公司，VN.py 提供了更稳定的 `CTPSTD` 连接方式：

```python
from vnpy.gateway.ctp import CtpstdGateway

# CTPSTD 连接
ctpstd_setting = {
    "用户名": "88888888",
    "密码": "password",
    "经纪商代码": "0081",
    "交易服务器": "121.37.81.82:41205",
    "行情服务器": "121.37.81.82:41213",
    "产品名称": "simnow_client",
    "授权编码": "0000000000000000",  # 仿真填16个0
}
```

## 4. 常见问题

### Q1: 连接超时/拒绝
- 检查防火墙和安全软件
- 确认服务器地址和端口正确
- 确认 Broker ID 正确

### Q2: 登录失败 (Error No. 3)
- 用户名或密码错误
- 期货账户未激活
- 交易日不在服务时间内

### Q3: 行情断线
- SimNow 行情不稳定是正常现象
- 实盘建议使用专线路连接
- 可开启自动重连

### Q4: 交易权限不足
- 确认账户已开通相应合约交易权限
- 部分合约需要单独申请
