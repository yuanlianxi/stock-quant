## 2. 策略实现

### 策略一：双均线策略（MA5/MA20）

**核心逻辑**：快线（MA5）上穿慢线（MA20）买入，下穿卖出。

```python
class DualMAStrategy(bt.Strategy):
    params = (
        ('fast', 5),   # 快线周期
        ('slow', 20),  # 慢线周期
    )

    def __init__(self):
        self.ma_fast = bt.indicators.SMA(self.data.close, period=self.params.fast)
        self.ma_slow = bt.indicators.SMA(self.data.close, period=self.params.slow)
        # 金叉/死叉信号
        self.crossover = bt.indicators.CrossOver(self.ma_fast, self.ma_slow)

    def next(self):
        if not self.position:
            # 无持仓，金叉买入
            if self.crossover > 0:
                self.buy()
        else:
            # 有持仓，死叉卖出
            if self.crossover < 0:
                self.close()
```

### 策略二：布林带策略（BB）

**核心逻辑**：价格触及下轨买入，触及上轨卖出。

```python
class BollingerBandsStrategy(bt.Strategy):
    params = (
        ('period', 20),
        ('devfactor', 2.0),
    )

    def __init__(self):
        self.boll = bt.indicators.BollingerBands(
            self.data.close,
            period=self.params.period,
            devfactor=self.params.devfactor
        )

    def next(self):
        if not self.position:
            if self.data.close < self.boll.lines.bot:
                self.buy()
        else:
            if self.data.close > self.boll.lines.top:
                self.close()
```

### 策略四：网格交易策略

**核心逻辑**：在固定价格区间设置网格，低买高卖，适合震荡市场。

```python
class GridTradingStrategy(bt.Strategy):
    params = (
        ('grid_num', 10),       # 网格数量
        ('grid_size', 0.02),    # 每格大小 2%
        ('upper_bound', None),
        ('lower_bound', None),
    )

    def __init__(self):
        # 动态计算上下边界
        self.upper = self.params.upper_bound or self.data.close[0] * 1.1
        self.lower = self.params.lower_bound or self.data.close[0] * 0.9
        self.grid_step = (self.upper - self.lower) / self.params.grid_num
        self.last_grid = None

    def next(self):
        price = self.data.close[0]
        current_grid = int((price - self.lower) / self.grid_step)

        if not self.position:
            # 价格触及下层网格买入
            if price <= self.lower + self.grid_step * 2:
                self.buy()
                self.last_grid = current_grid
        else:
            # 价格触及上层网格卖出
            if price >= self.upper - self.grid_step * 2:
                self.close()
                self.last_grid = None
```

### 策略五：海龟交易策略（Turtle Trading）

**核心逻辑**：趋势跟踪经典系统，突破20日最高价做多，跌破10日最低价平仓。用ATR管理仓位和止损。

```python
class TurtleStrategy(bt.Strategy):
    params = (
        ('entry_period', 20),    # 入场周期（突破20日最高做多）
        ('exit_period', 10),      # 出场周期（跌破10日最低平仓）
        ('atr_period', 14),       # ATR周期
        ('risk_ratio', 0.02),    # 单次交易风险 2%
        ('max_units', 4),         # 最大持仓Unit数
        ('atr_position', 0.5),    # 加仓间隔（0.5ATR）
    )

    def __init__(self):
        # 指标计算
        self.highest = bt.indicators.Highest(self.data.high, period=self.params.entry_period)
        self.lowest = bt.indicators.Lowest(self.data.low, period=self.params.entry_period)
        self.exit_stop = bt.indicators.Lowest(self.data.low, period=self.params.exit_period)
        self.atr = bt.indicators.ATR(self.data, period=self.params.atr_period)
        
        self.order = None
        self.units = 0
        self.entry_price = None

    def next(self):
        if self.order:
            self.order = None
        
        if not self.position:
            # 无持仓，检查是否突破
            if self.data.close[0] > self.highest[-1]:
                self.entry_price = self.data.close[0]
                # 计算仓位
                risk = self.broker.getvalue() * self.params.risk_ratio
                size = int(risk / (self.atr[0] * 100))  # 每手100股
                if size > 0:
                    self.order = self.buy(size=size)
                    self.units = 1
        else:
            # 有持仓，检查加仓和止损
            if self.units < self.params.max_units:
                # 加仓条件：价格继续上涨0.5ATR
                if self.data.close[0] > self.entry_price + self.params.atr_position * self.atr[0]:
                    risk = self.broker.getvalue() * self.params.risk_ratio
                    size = int(risk / (self.atr[0] * 100))
                    if size > 0:
                        self.order = self.buy(size=size)
                        self.units += 1
            
            # 止损：跌破10日最低或亏损2ATR
            stop_price = self.data.close[0] < self.exit_stop[-1]
            loss_stop = self.data.close[0] < self.entry_price - 2 * self.atr[0]
            if stop_price or loss_stop:
                self.order = self.close()
                self.units = 0
```

**海龟策略核心参数表**

| 参数 | 值 | 说明 |
|------|-----|------|
| 入场周期 | 20日 | 突破20日最高价做多 |
| 出场周期 | 10日 | 跌破10日最低价平仓 |
| 加仓间隔 | 0.5 ATR | 每上涨0.5倍ATR加仓 |
| 最大持仓 | 4 Unit | 最多加仓4次 |
| 止损 | 2 ATR | 亏损2ATR全平 |
| 单次风险 | 账户 2% | 每笔交易最大亏损 |

### Unit 计算公式详解

**目的**：根据账户风险和品种波动率计算每笔交易的手数

**公式**：
```
Unit = 账户风险金额 / (ATR × 每手合约乘数)

其中：
- 账户风险金额 = 账户总额 × 2%（单笔最大风险）
- ATR = 14日平均真实波幅
- 每手合约乘数 = 股票每手100股，期货按品种定义
```

**示例**（沪深300ETF）：
```
账户总额：100万
ATR（14日）：0.15元
每手合约乘数：100股

账户风险 = 1000000 × 2% = 20000元
Unit = 20000 / (0.15 × 100) = 1333股 ≈ 13手（每手100股）
```

**示例**（螺纹钢期货）：
```
账户总额：100万
ATR（14日）：50元
每手合约乘数：10吨

账户风险 = 1000000 × 2% = 20000元
Unit = 20000 / (50 × 10) = 40手
```

## 7. 更多交易策略技术详解

### 7.1 趋势跟踪策略

#### RSI 顺势策略
- **原理**：RSI 超卖后向上突破70视为强势信号
- **参数**：RSI周期=14，阈值=70/30
- **代码**：
```python
class RSITrendStrategy(bt.Strategy):
    params = (('rsi_period', 14), ('overbought', 70), ('oversold', 30))
    
    def __init__(self):
        self.rsi = bt.indicators.RSI(self.data.close, period=self.params.rsi_period)
    
    def next(self):
        if not self.position:
            if self.rsi[0] > self.params.overbought:
                self.buy()
        else:
            if self.rsi[0] < self.params.oversold:
                self.close()
```
- **适用场景**：强势趋势确认后的顺势交易，适合趋势明显的市场

#### KDJ 金叉策略
- **原理**：K线上穿D线（金叉）+ 超卖区间买入
- **参数**：N=9，M1=3，M2=3
- **代码**：
```python
class KDJStrategy(bt.Strategy):
    params = (('period', 9), ('smooth_k', 3), ('smooth_d', 3))
    
    def __init__(self):
        self.k = bt.indicators.Stochastic(self.data, period=self.params.period, 
                                          smooth_d=self.params.smooth_d)
    
    def next(self):
        if not self.position:
            if self.k.lines.k[0] > self.k.lines.d[0] and self.k.lines.k[0] < 30:
                self.buy()
        else:
            if self.k.lines.k[0] < self.k.lines.d[0] and self.k.lines.k[0] > 70:
                self.close()
```
- **适用场景**：震荡市场中的超买超卖信号，需结合趋势过滤避免逆势交易

### 7.2 均值回归策略

#### 布林带均值回归
- **原理**：价格触及下轨后回归均值的概率高
- **参数**：MA20，2倍标准差
- **代码**：
```python
class BollingerMeanReversion(bt.Strategy):
    params = (('period', 20), ('devfactor', 2.0), ('hold_days', 5))
    
    def __init__(self):
        self.boll = bt.indicators.BollingerBands(self.data.close, 
                                                  period=self.params.period,
                                                  devfactor=self.params.devfactor)
        self.orders = {}
    
    def next(self):
        if not self.position:
            # 下穿下轨且RSI超卖
            if self.data.close[0] < self.boll.lines.bot[0]:
                self.buy()
                self.order_days = 0
        else:
            self.order_days += 1
            # 触及中轨或持有满N天强制平仓
            if self.data.close[0] >= self.boll.lines.mid[0] or self.order_days >= self.params.hold_days:
                self.close()
```
- **适用场景**：震荡市场，价格触及布林下轨后反弹的概率较高

#### CCI 超卖策略
- **原理**：CCI < -100 视为超卖，反弹概率高
- **参数**：CCI周期=14
- **代码**：
```python
class CCIStrategy(bt.Strategy):
    params = (('period', 14), ('oversold', -100), ('overbought', 100))
    
    def __init__(self):
        self.cci = bt.indicators.CommodityChannelIndex(self.data, period=self.params.period)
    
    def next(self):
        if not self.position:
            if self.cci[0] < self.params.oversold:
                self.buy()
        else:
            if self.cci[0] > self.params.overbought:
                self.close()
```
- **适用场景**：商品期货和股票的超买超卖信号，CCI 对价格变化的敏感度高于 RSI

### 7.3 突破策略

#### 区间突破策略
- **原理**：价格突破过去N日最高点时买入
- **参数**：周期=20，仓位=10%
```python
class BreakoutStrategy(bt.Strategy):
    params = (('period', 20), ('position_size', 0.1))
    
    def __init__(self):
        self.highest = bt.indicators.Highest(self.data.high, period=self.params.period)
    
    def next(self):
        if not self.position:
            if self.data.close[0] > self.highest[-1]:
                size = int(self.broker.getvalue() * self.params.position_size / self.data.close[0])
                self.buy(size=size)
        else:
            # 止损：跌破入场价5%
            if self.data.close[0] < self.position.price * 0.95:
                self.close()
```
- **适用场景**：趋势行情启动初期，适合大盘指数 ETF

#### ATR 止损的突破策略
- **原理**：突破入场 + ATR 动态止损
```python
class ATRBreakoutStrategy(bt.Strategy):
    params = (('period', 20), ('atr_period', 14), ('atr_multiplier', 2))
    
    def __init__(self):
        self.highest = bt.indicators.Highest(self.data.high, period=self.params.period)
        self.atr = bt.indicators.ATR(self.data, period=self.params.atr_period)
        self.entry_price = None
    
    def next(self):
        if not self.position:
            if self.data.close[0] > self.highest[-1]:
                self.entry_price = self.data.close[0]
                self.buy()
        else:
            # ATR止损
            stop_price = self.entry_price - self.params.atr_multiplier * self.atr[0]
            if self.data.close[0] < stop_price:
                self.close()
```
- **适用场景**：趋势行情中的动态止损，避免固定止损被频繁扫出

### 7.4 通道策略

#### 唐奇安通道策略
- **原理**：突破上轨买入，跌破下轨卖出
- **参数**：通道周期=20
```python
class DonchianChannelStrategy(bt.Strategy):
    params = (('period', 20))
    
    def __init__(self):
        self.upper = bt.indicators.Highest(self.data.high, period=self.params.period)
        self.lower = bt.indicators.Lowest(self.data.low, period=self.params.period)
        self.exit_upper = bt.indicators.Lowest(self.data.low, period=int(self.params.period/2))
    
    def next(self):
        if not self.position:
            if self.data.close[0] > self.upper[-1]:
                self.buy()
        else:
            if self.data.close[0] < self.exit_upper[-1]:
                self.close()
```
- **适用场景**：趋势跟踪经典方法，出场信号更快，适合短期趋势

#### 肯特纳通道策略
- **原理**：基于 ATR 的波动率通道，突破上轨买入
```python
class KeltnerChannelStrategy(bt.Strategy):
    params = (('period', 20), ('atr_period', 10), ('atr_multiplier', 2))
    
    def __init__(self):
        self.sma = bt.indicators.SMA(self.data.close, period=self.params.period)
        self.atr = bt.indicators.ATR(self.data, period=self.params.atr_period)
        self.upper = self.sma + self.params.atr_multiplier * self.atr
        self.lower = self.sma - self.params.atr_multiplier * self.atr
    
    def next(self):
        if not self.position:
            if self.data.close[0] > self.upper[0]:
                self.buy()
        else:
            if self.data.close[0] < self.lower[0]:
                self.close()
```
- **适用场景**：波动率自适应通道，比固定通道更能反映市场实际波动水平

### 7.5 量价策略

#### 放量突破策略
- **原理**：成交量放大 + 价格突破 = 趋势确认
```python
class VolumeBreakoutStrategy(bt.Strategy):
    params = (('volume_multiplier', 1.5), ('price_period', 20))
    
    def __init__(self):
        self.sma = bt.indicators.SMA(self.data.close, period=self.params.price_period)
        self.volume_sma = bt.indicators.SMA(self.data.volume, period=self.params.price_period)
    
    def next(self):
        volume_ratio = self.data.volume[0] / self.volume_sma[0] if self.volume_sma[0] > 0 else 0
        price_breakout = self.data.close[0] > self.sma[0]
        
        if not self.position:
            if volume_ratio > self.params.volume_multiplier and price_breakout:
                self.buy()
        else:
            if self.data.close[0] < self.sma[0]:
                self.close()
```
- **适用场景**：需要趋势确认的场景，放量是趋势启动的重要信号

#### 缩量回调策略

```python
class VolumePullbackStrategy(bt.Strategy):

                self.buy()
```

### 7.7 综合策略模板

#### 多指标确认策略
```python
class MultiIndicatorStrategy(bt.Strategy):
    params = (
        ('ma_fast', 5), ('ma_slow', 20),
        ('rsi_period', 14), ('rsi_overbought', 70),
        ('atr_period', 14),
    )
    
    def __init__(self):
        self.ma_fast = bt.indicators.SMA(self.data.close, period=self.params.ma_fast)
        self.ma_slow = bt.indicators.SMA(self.data.close, period=self.params.ma_slow)
        self.rsi = bt.indicators.RSI(self.data.close, period=self.params.rsi_period)
        self.atr = bt.indicators.ATR(self.data, period=self.params.atr_period)
    
    def next(self):
        trend = self.ma_fast[0] > self.ma_slow[0]
        momentum = self.rsi[0] < self.params.rsi_overbought
        confluence = trend and momentum
        
        if not self.position:
            if confluence:
                self.buy()
        else:
            # ATR跟踪止损
            stop = self.position.price - 2 * self.atr[0]
            if self.data.close[0] < stop:
                self.close()
```
- **适用场景**：多指标共振确认，提高信号可靠性，减少假信号
