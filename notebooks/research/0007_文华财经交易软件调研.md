# 文华财经交易软件调研

> 版本：v1.0

## 1. 产品概述

文华财经是国内期货行业主流程序化交易软件提供商，拥有多条产品线，覆盖个人投资者至机构投资者。

| 产品 | 定位 | 目标用户 |
|------|------|---------|
| **WH8**（wt8） | 趋势策略 + 自动交易 | 个人投资者、量化入门者 |
| **WT9**（宽语言） | 量化策略开发平台 | 专业量化机构、金工团队 |
| **WH6** | 基础行情+交易 | 普通投资者 |
| **随身行** | 移动端交易 | 移动交易用户 |

---

## 2. WH8 功能详解

### 2.1 定位
基于麦语言的量化策略开发平台，采用积木式编程环境，简单易用，适合入门。

### 2.2 核心功能

| 功能 | 说明 |
|------|------|
| **麦语言编程** | 积木式指标拼装，语法简单 |
| **趋势策略开发** | 支持自定义技术指标、交易条件 |
| **自动交易** | 条件触发自动下单 |
| **历史数据回测** | 提供国内期货全品种历史数据 |
| **期货交易接口** | 直连期货公司CTP |

### 2.3 特点

- 积木式编程，上手快
- 内置丰富指标库
- 适合简单趋势跟踪策略
- 封闭生态，策略安全性依赖文华平台

---

## 3. WT9（宽语言）功能详解

### 3.1 定位
基于宽语言（C语言风格）的量化策略开发平台，支持复杂策略、订单流、算法交易，服务于专业量化机构。

### 3.2 核心功能

#### 策略研发

| 功能 | 说明 |
|------|------|
| **趋势策略研发** | 自定义指标组合、多周期联动 |
| **订单流策略研发** | 基于盘口挂单数据建模 |
| **算法交易策略研发** | 智能拆单、降低冲击成本 |

#### 回测引擎

| 功能 | 说明 |
|------|------|
| **超长历史数据** | 国内期货全品种上市以来全部历史数据 |
| **Tick级数据回测** | 精准模拟真实成交 |
| **主连链回测** | 规避换月跳空，专用回测算法 |
| **360度回测报告** | 上百项统计指标 |
| **可视化回测平台** | 回测过程同步展示行情、挂单、持仓 |
| **仿真撮合机制** | 挂单不直接成交，按挂单成交比例撮合 |

#### 交易执行

| 功能 | 说明 |
|------|------|
| **模型信号算法交易** | 信号处理+算法拆单，提高交易效率 |
| **手动下单算法交易** | 手动补单也支持智能拆单 |
| **算法下单组件** | 多种算法（VWAP/TWAP/滑点控制等）|

#### 组合管理

| 功能 | 说明 |
|------|------|
| **组合配置** | 多模型组合、相关性分析 |
| **组合参数优化** | 遗传规划算法自动定参 |
| **资金管理** | 头寸管理函数、申购赎回管理 |
| **收益分析** | 分区运行单元独立核算 |
| **模型运行监控** | 集中管理多模型运行状态 |
| **模型加密运行** | 本地运行，源码保护 |

## 5. 与我们项目的对比

| 维度 | 文华财经 WH8/WT9 | 我们的回测系统 |
|------|-----------------|---------------|
| **策略开发** | 专属语言，封闭生态 | Python，完全可控 |
| **数据回测** | 内置数据，Tick级 | 自建 DataLoader |
| **实盘对接** | 直接支持（CTP）| 后续 VN.py 对接 |
| **费用** | 收费，价格较高 | 开源免费 |
| **灵活性** | 受限于平台功能 | 可深度定制 |
| **订单流分析** | WT9 支持 | 需自研 |

## 6. 适用场景建议

### 6.1 适合使用文华财经

- 追求快速上线，不想自己搭建系统
- 策略逻辑简单（技术指标组合）
- 需要直接对接实盘（文华已集成CTP）
- 个人投资者，成本敏感度低

### 6.2 适合自建系统（我们）

- 需要深度定制策略逻辑
- 想积累量化研发能力
- 策略需要与机器学习/AI结合
- 开源可控，成本敏感
- 需要与现有系统集成

## 7. 总结

| 产品 | 推荐度 | 适用场景 |
|------|--------|---------|
| **WH8** | ⭐⭐⭐⭐ | 入门量化，不想写代码 |
| **WT9** | ⭐⭐⭐⭐⭐ | 专业量化，有C语言基础 |
| **自建系统** | ⭐⭐⭐⭐⭐ | 长期投入，想深度自研 |

**推荐路径**：
```

自研阶段：Python + Backtrader（灵活可控）
实盘阶段：VN.py（开源对接CTP）
```
