## 1. 策略起源与背景

### 1.1 传奇故事

海龟交易策略（ Turtle Trading）源于 1983 年芝加哥期货交易大师 **理查德·丹尼斯（Richard Dennis）** 与他的好友 **威廉·埃克哈特（William Eckhardt）** 之间的一场著名赌约。

丹尼斯认为，交易技巧可以通过后天教导习得，而埃克哈特则认为交易是一种天赋，是骨子里的东西，谁也教不会。

为了验证这个争论，丹尼斯招募了 23 名普通人作为"海龟"（Turtles），对他们进行了为期两周的密集培训，然后给了他们真实资金进行交易。这些"海龟学员"后来在 4 年内创造了超过 1.75 亿美元的利润，证明了丹尼斯的观点——交易可以被教导。

### 1.2 策略公开

1984 年，斯坦利·科克兰（Stanley Kroll）在《 New Commodities Trading System》一书中首次公开了海龟交易策略的核心思路。1990 年，另一位海龟成员 **Curtis Faith** 出版了《 Way of the Turtle》，详细披露了完整策略规则，使得这一系统得以广泛流传。

### 1.3 策略本质

海龟交易策略是一个**趋势跟踪系统**，它的核心思想是：

- **截断亏损，让利润奔跑**（Cut loss short, let profits run）
- **顺势而为**，在趋势开始时入场，在趋势结束时出场
- 通过**严格的仓位管理**控制风险，通过**分散投资**降低波动

### 1.4 原版海龟 vs 改良海龟

| 版本 | 入场周期 | 出场周期 | 适用市场 |
|------|---------|---------|---------|
| 原版（28日） | 28 日突破 | 20 日最低价 | 期货为主 |
| 经典（20/10） | 20 日突破 | 10 日最低价 | 股票/ETF |
| A 股改良版 | 18-20 日 | 10-12 日 | A 股 |

## 2. 核心理念与原则

### 2.1 五大核心原则

**原则一：市场是友好的**
交易者不应与市场对抗。当市场证明你错的时候，承认它；当市场证明你对的时候，相信它。

**原则二：纪律重于预测**
不依赖预测或直觉，而是依赖一套规则系统。规则是客观的，执行是纪律的。

**原则三：风险控制是第一要务**
每笔交易的风险必须严格控制在总资金的 1%-2% 以内。使用 ATR 动态计算仓位，确保所有品种风险均等。

**原则四：分散投资降低风险**
同时交易多个不相关的品种，可以降低整体账户的波动性。海龟法则建议同时持有 4-6 个品种。

**原则五：耐心等待高概率机会**
不是每天都要交易。只有当价格突破 20 日（经典）或 28 日（原版）高点时，才考虑入场。

### 2.2 交易者的四种境界

| 境界 | 描述 | 状态 |
|------|------|------|
| 第一层 | 不知道交易需要规则 | 亏损 |
| 第二层 | 知道规则但无法坚持 | 亏损 |
| 第三层 | 坚持规则但担心执行 | 小盈 |
| 第四层 | 规则即本能 | 稳定盈利 |

### 2.3 心理偏差与克服

海龟策略的设计充分考虑了人类心理偏差：

| 心理偏差 | 表现 | 海龟法则的应对 |
|---------|------|---------------|
| 损失厌恶 | 亏损时死扛 | 硬止损 2ATR |
| 近期偏好 | 过度重视近期数据 | 固定周期规则 |
| 确认偏误 | 只看支持自己的信息 | 客观的突破规则 |
| 过度自信 | 重仓交易 | 2% 风险上限 |
| 后悔厌恶 | 错过机会后追高 | 规则驱动，不追 |

## 3. 完整交易规则详解

### 3.1 基础数据要求

在开始交易之前，需要准备以下数据：

```
1. 每日最高价（High）
2. 每日最低价（Low）
3. 每日收盘价（Close）
4. 持仓量/成交量（Volume，可选）
```

### 3.2 入场规则（Entry）

**做多入场条件**：
1. 当前收盘价 > 过去 N 日的最高价
2. N 通常为 20（原版为 55 或 28）

**做空入场条件**（期货/可做空品种）：
1. 当前收盘价 < 过去 N 日的最低价
2. 本文档以做多为主

**入场流程**：
```
信号产生 → 计算仓位 → 下单 → 记录入场价和 Unit 数
```

### 3.3 加仓规则（Add to Position）

加仓条件（同时满足）：
1. 当前有持仓（未触发止损）
2. 价格继续上涨，从最近一次加仓价格算起，已上涨 ≥ 0.5 ATR
3. 当前持仓 Unit 数 < 允许的最大 Unit 数

**加仓上限**：
- 原版海龟：最多 4 个 Unit
- A 股改良版：最多 3-4 个 Unit
- 建议根据策略自信度调整

**加仓公式**：
```
加仓价格 = 最后一次加仓价格 + (0.5 × ATR)
新加仓手数 = 账户风险 / (ATR × 合约乘数)
```

### 3.4 止损规则（Stop Loss）

**止损条件**（满足任一即触发）：
1. 价格跌破最后 N1 日最低价（原版 N1 = 10）
2. 价格跌破入场价 - 2 ATR

**止损特点**：
- **跟踪止损**：随着价格继续上涨，止损价格也在上移
- **绝对止损**：最大亏损不超过入场价的 2 ATR

**止损后的处理**：
1. 平仓所有持仓
2. 该品种进入"禁止交易期"（原版为 20-60 日）
3. 等待下一次突破信号

### 3.5 出场规则（Exit）

**出场条件**：
1. 价格跌破过去 N2 日的最低价（N2 通常为 10）
2. 触发止损（见 3.4）
3. 达到最大持仓限制后的反向信号

**出场流程**：
```
信号产生 → 平仓 → 记录盈亏 → 更新账户状态
```

### 3.6 完整交易周期图

```
入场信号 → 做多 → 加仓1 → 加仓2 → 加仓3 → 加仓4 → 出场信号 → 平仓
   ↑                                              ↓
   ←←←←←←←←←←  止损触发（任意环节）  ←←←←←←←←←←←←←←←←
```

## 4. Unit 系统与仓位管理

### 4.1 Unit 的定义

**Unit** 是海龟交易系统的核心仓位单位，代表**根据市场波动率调整后的标准交易量**。

Unit 的设计目的是：让不同波动率的品种对账户的风险贡献大致相同。

### 4.2 Unit 计算公式

```
Unit = 账户风险金额 / (ATR × 每手合约乘数)

其中：
- 账户风险金额 = 账户总额 × 2%（单笔最大风险）
- ATR = N 日平均真实波幅（通常 14 日）
- 每手合约乘数 = 股票每手 100 股，期货按品种定义
```

### 4.3 公式推导

设：
- 账户总额 = A
- 单次风险比例 = r（通常 2%）
- ATR = 平均真实波幅
- 合约乘数 = M（股票为 100，期货按品种）

则：
```
风险金额 = A × r
每手风险 = ATR × M
Unit = (A × r) / (ATR × M)
```

### 4.4 计算示例

**示例一：沪深 300ETF（510300）**

```
账户总额：1,000,000 元
单次风险比例：2%
ATR（14日）：0.15 元
每手合约乘数：100 股

计算：
账户风险 = 1,000,000 × 2% = 20,000 元
每手风险 = 0.15 × 100 = 15 元
Unit = 20,000 / 15 = 1,333 股

实际下单：约 13 手（每手 100 股）
```

**示例二：螺纹钢期货（RB）**

```
账户总额：1,000,000 元
单次风险比例：2%
ATR（14日）：50 元
每手合约乘数：10 吨/手

计算：
账户风险 = 1,000,000 × 2% = 20,000 元
每手风险 = 50 × 10 = 500 元
Unit = 20,000 / 500 = 40 手
```

**示例三：原油期货（SC）**

```
账户总额：1,000,000 元
单次风险比例：2%
ATR（14日）：3.5 美元
每手合约乘数：1000 桶/手

计算：
账户风险 = 1,000,000 × 2% = 20,000 元（约 2,800 美元，汇率 7.1）
每手风险 = 3.5 × 1000 = 3,500 美元
Unit = 2,800 / 3,500 ≈ 0.8 手

结论：原油期货波动大，单次只能交易不到 1 手
```

### 4.5 多 Unit 持仓的风险分析

| 持仓 Unit 数 | 占用保证金（估算）| 账户风险占比 | 最大亏损 |
|-------------|-----------------|-------------|---------|
| 1 Unit | 约 5-10% | 2% | 2% |
| 2 Unit | 约 10-20% | 4% | 4% |
| 3 Unit | 约 15-30% | 6% | 6% |
| 4 Unit | 约 20-40% | 8% | 8% |

**注意**：上述风险占比为理论值，实际亏损可能因加仓位置不同而略有差异。

### 4.6 仓位管理原则

1. **单一品种最大风险不超过账户的 2% × 4 = 8%**
2. **所有持仓总风险控制在账户的 20% 以内**
3. **同时交易品种数建议 4-6 个**
4. **每个品种的持仓不超过总风险的 1/4**

## 5. ATR 计算方法

### 5.1 真实波幅（True Range, TR）

真实波幅是以下三者中的最大值：

```
TR = max(今日最高价 - 今日最低价,
          今日最高价 - 昨日收盘价,
          今日最低价 - 昨日收盘价)
```

**意义**：真实波幅考虑了跳空开盘的情况，比简单的最高-最低价差更准确地反映市场波动。

### 5.2 平均真实波幅（ATR）

 ATR 是 TR 的移动平均值：

```
ATR(N) = (ATR(N-1) × (N-1) + 今日 TR) / N
```

也可以用指数移动平均（EMA）计算：

```
ATR(N) = EMA(TR, N)
```

**常用 N 值**：
- 原版海龟：20 日
- 经典用法：14 日
- A 股改良：10-20 日

### 5.3 ATR 的含义

 ATR 的数值表示：**过去 N 日市场平均波动幅度（以价格单位计）**

解读示例：
- ATR = 0.15 元 → 过去 14 个交易日，平均每天波动 0.15 元
- ATR = 50 点 → 过去 14 个交易日，平均每天波动 50 个点

### 5.4 ATR 在海龟策略中的应用

| 用途 | 公式 | 说明 |
|------|------|------|
| 仓位计算 | Unit = 风险金额 / (ATR × 乘数) | 根据波动率调整仓位 |
| 加仓间隔 | 0.5 ATR | 价格每上涨 0.5 ATR 加仓一次 |
| 止损距离 | 2 ATR | 从入场价最多亏损 2 ATR |
| 止损价格 | 入场价 - 2 ATR | 硬止损价格 |

### 5.5 Python 实现

```python
import pandas as pd
import numpy as np

def calculate_tr(high, low, close_prev):
    """
    计算真实波幅（True Range）
    """
    h_l = high - low  # 当日高低差
    h_c = abs(high - close_prev)  # 最高-前收盘
    l_c = abs(low - close_prev)  # 最低-前收盘
    return np.maximum(h_l, np.maximum(h_c, l_c))

def calculate_atr(df, period=14):
    """
    计算平均真实波幅（ATR）
    
    参数：
        df: DataFrame，需要包含 high, low, close 列
        period: ATR 周期，默认 14
    返回：
        ATR Series
    """
    df = df.copy()
    
    # 计算真实波幅
    df['tr'] = calculate_tr(
        df['high'], 
        df['low'], 
        df['close'].shift(1)
    )
    
    # 计算 ATR（使用指数移动平均）
    df['atr'] = df['tr'].ewm(alpha=1/period, min_periods=period).mean()
    
    return df['atr']

# 使用示例
# df = pd.read_csv('stock_data.csv')
# df['atr_14'] = calculate_atr(df, period=14)
```

## 6. 入场、出场、加仓、止损规则

### 6.1 入场规则详解

**做多入场**：

| 步骤 | 操作 | 代码逻辑 |
|------|------|---------|
| 1 | 计算 N 日最高价 | `highest = pd.rolling_max(df['high'], N)` |
| 2 | 比较收盘价与最高价 | `signal = df['close'] > highest.shift(1)` |
| 3 | 确认无持仓 | `position == 0` |
| 4 | 下单 | `buy(size=unit)` |

**关键点**：
- 用 `shift(1)` 是因为不能使用当日最高价（作弊）
- 必须等收盘后确认信号，避免日内假突破

**入场伪代码**：

```python
def check_entry(df, position, params):
    if position > 0:
        return None  # 已有持仓，不入场
    
    entry_price = df['close'].iloc[-1]
    highest_n = df['high'].rolling(params.entry_period).max().iloc[-2]  # 前一日 N 日最高
    
    if entry_price > highest_n:
        return {
            'action': 'BUY',
            'price': entry_price,
            'size': calculate_unit(account_value, df['atr'].iloc[-1], params)
        }
    return None
```

### 6.2 加仓规则详解

**加仓条件**（必须同时满足）：

| 条件 | 说明 |
|------|------|
| 已有持仓 | position > 0 |
| 未触发止损 | 当前价格 > 止损价 |
| 价格已上涨 | 当前价格 ≥ 最后加仓价 + params.add_interval × ATR |
| 未达上限 | current_units < params.max_units |

**加仓伪代码**：

```python
def check_add(df, position, last_add_price, current_units, params):
    if position == 0:
        return None  # 无持仓，不加仓
    
    current_price = df['close'].iloc[-1]
    atr = df['atr'].iloc[-1]
    add_threshold = last_add_price + params.add_interval * atr
    
    if (current_price >= add_threshold and 
        current_units < params.max_units):
        return {
            'action': 'BUY',
            'price': current_price,
            'size': calculate_unit(account_value, atr, params)
        }
    return None
```

### 6.3 止损规则详解

**止损触发条件**（满足任一）：

| 条件 | 说明 | 计算公式 |
|------|------|---------|
| 跌破 N 日最低 | 短期趋势反转 | stop_price = lowest_n |
| 跌破入场价 - 2ATR | 超过最大容忍亏损 | stop_price = entry_price - 2 × atr |

**止损逻辑**：

```python
def check_stop(df, position, entry_price, atr, params):
    if position == 0:
        return None  # 无持仓，不止损
    
    current_price = df['close'].iloc[-1]
    
    # 计算止损价
    stop_by_lowest = df['low'].rolling(params.exit_period).min().iloc[-2]
    stop_by_atr = entry_price - params.stop_loss * atr
    
    # 取两者较高值（更保守）
    stop_price = max(stop_by_lowest, stop_by_atr)
    
    if current_price <= stop_price:
        return {
            'action': 'SELL',
            'reason': 'STOP_LOSS',
            'price': current_price
        }
    return None
```

### 6.4 出场规则详解

**出场条件**：

| 条件 | 说明 |
|------|------|
| 跌破 N 日最低 | 趋势结束 |
| 触发止损 | 风险控制 |

**出场伪代码**：

```python
def check_exit(df, position, entry_price, atr, params):
    if position == 0:
        return None  # 无持仓，不出场
    
    current_price = df['close'].iloc[-1]
    lowest_n = df['low'].rolling(params.exit_period).min().iloc[-2]
    
    if current_price <= lowest_n:
        return {
            'action': 'SELL',
            'reason': 'EXIT_SIGNAL',
            'price': current_price
        }
    return None
```

### 7.1 核心参数表

**方法一：网格搜索**

```python
from itertools import product
import numpy as np

def grid_search(df, param_grid):
    """
    网格搜索最优参数
    
    param_grid = {
        'entry_period': [18, 20, 22, 25],
        'exit_period': [10, 12, 15],
        'atr_period': [14, 20],
        'max_units': [3, 4],
        'stop_loss': [2],
        'add_interval': [0.5]
    }
    """
    results = []
    
    # 生成所有参数组合
    keys, values = zip(*param_grid.items())
    for combo in product(*values):
        params = dict(zip(keys, combo))
        
        # 运行回测
        result = backtest(df, params)
        results.append({
            'params': params,
            'sharpe': result['sharpe_ratio'],
            'return': result['annual_return'],
            'max_drawdown': result['max_drawdown']
        })
    
    # 按夏普比率排序
    results.sort(key=lambda x: x['sharpe'], reverse=True)
    return results
```

**方法二：Walk-Forward 分析**

避免过拟合的推荐方法：

```
1. 将数据分为：训练期（70%）+ 测试期（30%）
2. 在训练期用网格搜索找到最优参数
3. 在测试期验证参数效果
4. 重复多次（滚动窗口）
5. 取各窗口平均值作为最终参数
```

```python
import backtrader as bt
import pandas as pd
import numpy as np

class TurtleStrategy(bt.Strategy):
    """
    海龟交易策略
    
    入场：突破 N 日最高价做多
    出场：跌破 M 日最低价平仓
    加仓：每上涨 0.5ATR 加仓一次
    止损：亏损 2ATR 全平
    """
    
    params = (
        ('entry_period', 20),      # 入场周期
        ('exit_period', 10),       # 出场周期
        ('atr_period', 14),        # ATR 周期
        ('risk_ratio', 0.02),      # 单次风险比例 2%
        ('max_units', 4),           # 最大持仓 Unit 数
        ('add_interval', 0.5),     # 加仓间隔（ATR 倍数）
        ('stop_loss', 2.0),         # 止损倍数（ATR）
    )
    
    def __init__(self):
        # 指标计算
        self.highest = bt.indicators.Highest(
            self.data.high, 
            period=self.params.entry_period
        )
        self.lowest = bt.indicators.Lowest(
            self.data.low, 
            period=self.params.exit_period
        )
        self.atr = bt.indicators.ATR(
            self.data, 
            period=self.params.atr_period
        )
        
        # 状态跟踪
        self.order = None
        self.units = 0  # 当前持仓 Unit 数
        self.entry_price = 0  # 入场价格
        self.last_add_price = 0  # 最后一次加仓价格
        self.trade_log = []  # 交易日志
        
    def log(self, txt, dt=None):
        """日志输出"""
        dt = dt or self.datas[0].datetime.date(0)
        print(f'[{dt.isoformat()}] {txt}')
    
    def notify_order(self, order):
        """订单通知"""
        if order.status in [order.Submitted, order.Accepted]:
            return
        
        if order.status in [order.Completed]:
            if order.isbuy():
                self.log(f'买入执行: 价格 {order.executed.price:.2f}, '
                         f'数量 {order.executed.size}, '
                         f'成本 {order.executed.value:.2f}')
            else:
                self.log(f'卖出执行: 价格 {order.executed.price:.2f}, '
                         f'数量 {order.executed.size}')
            self.order = None
        elif order.status in [order.Canceled, order.Margin, order.Rejected]:
            self.log('订单被取消/保证金不足/拒绝')
            self.order = None
    
    def notify_trade(self, trade):
        """交易通知"""
        if trade.isclosed:
            self.log(f'交易平仓: 毛利 {trade.pnl:.2f}, 净利 {trade.pnlcomm:.2f}')
            self.trade_log.append({
                'entry': trade.entry_price,
                'exit': trade.exit_price,
                'pnl': trade.pnl,
                'pnlcomm': trade.pnlcomm,
                'bars': trade.barlen
            })
    
    def next(self):
        """主循环"""
        # 跳过未完成订单
        if self.order:
            return
        
        current_price = self.data.close[0]
        atr_value = self.atr[0]
        
        # ========== 无持仓：检查入场 ==========
        if not self.position:
            # 入场条件：价格突破 N 日最高
            if current_price > self.highest[-1]:
                self.entry_price = current_price
                self.last_add_price = current_price
                
                # 计算仓位
                risk_amount = self.broker.getvalue() * self.params.risk_ratio
                size = self._calculate_position_size(atr_value)
                
                if size > 0:
                    self.order = self.buy(size=size)
                    self.units = 1
                    self.log(f'入场信号: 突破 {self.params.entry_period} 日最高, '
                             f'价格 {current_price:.2f}, Unit {self.units}')
        
        # ========== 有持仓：检查加仓/止损/出场 ==========
        else:
            # ----- 检查加仓 -----
            if self.units < self.params.max_units:
                add_threshold = self.last_add_price + self.params.add_interval * atr_value
                if current_price >= add_threshold:
                    risk_amount = self.broker.getvalue() * self.params.risk_ratio
                    size = self._calculate_position_size(atr_value)
                    
                    if size > 0:
                        self.order = self.buy(size=size)
                        self.units += 1
                        self.last_add_price = current_price
                        self.log(f'加仓信号: 价格 {current_price:.2f}, '
                                 f'Unit {self.units}, 累计 {self.units} Unit')
            
            # ----- 检查止损 -----
            stop_price = self.entry_price - self.params.stop_loss * atr_value
            lowest_exit = self.lowest[-1]
            
            # 取更保守的止损价（两者中较高者）
            actual_stop = max(stop_price, lowest_exit)
            
            if current_price <= actual_stop:
                self.order = self.close()
                self.log(f'止损信号: 价格 {current_price:.2f}, '
                         f'止损价 {actual_stop:.2f}, Unit {self.units} -> 0')
                self.units = 0
            
            # ----- 检查出场 -----
            elif current_price <= lowest_exit:
                self.order = self.close()
                self.log(f'出场信号: 跌破 {self.params.exit_period} 日最低, '
                         f'价格 {current_price:.2f}, Unit {self.units} -> 0')
                self.units = 0
    
    def _calculate_position_size(self, atr_value):
        """计算仓位（Unit）"""
        risk_amount = self.broker.getvalue() * self.params.risk_ratio
        # 假设每手 100 股（股票），期货需调整合约乘数
        contract_multiplier = 100
        unit_size = int(risk_amount / (atr_value * contract_multiplier))
        return max(unit_size, 0)
    
    def stop(self):
        """策略结束时输出统计"""
        self.log('策略结束')


def run_backtest():
    """运行回测"""
    cerebro = bt.Cerebro()
    
    # 添加数据
    data = bt.feeds.PandasData(
        dataname=pd.read_csv('stock_data.csv', parse_dates=['date']),
        datetime='date',
        open='open',
        high='high',
        low='low',
        close='close',
        volume='volume',
        openinterest=-1
    )
    cerebro.adddata(data)
    
    # 设置初始资金
    cerebro.broker.setcash(1000000.0)
    
    # 设置手续费
    cerebro.broker.setcommission(commission=0.001)
    
    # 添加策略
    cerebro.addstrategy(TurtleStrategy)
    
    # 添加分析器
    cerebro.addanalyzer(bt.analyzers.SharpeRatio, _name='sharpe')
    cerebro.addanalyzer(bt.analyzers.Returns, _name='returns')
    cerebro.addanalyzer(bt.analyzers.DrawDown, _name='drawdown')
    cerebro.addanalyzer(bt.analyzers.TradeAnalyzer, _name='trades')
    
    # 运行回测
    results = cerebro.run()
    
    # 输出结果
    strat = results[0]
    print('\n========== 回测结果 ==========')
    print(f'最终资金: {cerebro.broker.getvalue():.2f}')
    print(f'夏普比率: {strat.analyzers.sharpe.get_analysis().get("sharperatio", 0):.3f}')
    print(f'年化收益: {strat.analyzers.returns.get_analysis().get("rnorm100", 0):.2f}%')
    print(f'最大回撤: {strat.analyzers.drawdown.get_analysis().get("max", {}).get("drawdown", 0):.2f}%')
    
    # 交易统计
    trade_analysis = strat.analyzers.trades.get_analysis()
    print(f'\n总交易次数: {trade_analysis.total.total}')
    print(f'盈利次数: {trade_analysis.pnl.gross.positive.count}')
    print(f'亏损次数: {trade_analysis.pnl.gross.negative.count}')
    
    if trade_analysis.pnl.gross.positive.count > 0:
        avg_win = trade_analysis.pnl.gross.positive.total / trade_analysis.pnl.gross.positive.count
        avg_loss = abs(trade_analysis.pnl.gross.negative.total / trade_analysis.pnl.gross.negative.count)
        print(f'平均盈利: {avg_win:.2f}')
        print(f'平均亏损: {avg_loss:.2f}')
        print(f'盈亏比: {avg_win/avg_loss:.2f}')


if __name__ == '__main__':
    run_backtest()
```

### 9.2 纯 Python 实现（无 Backtrader）

```python
import pandas as pd
import numpy as np
from typing import Dict, List, Optional, Tuple

class TurtleTradingSystem:
    """
    纯 Python 实现的海龟交易系统
    不依赖 Backtrader，适合自定义需求
    """
    
    def __init__(
        self,
        entry_period: int = 20,
        exit_period: int = 10,
        atr_period: int = 14,
        risk_ratio: float = 0.02,
        max_units: int = 4,
        add_interval: float = 0.5,
        stop_loss: float = 2.0,
        contract_multiplier: int = 100
    ):
        self.entry_period = entry_period
        self.exit_period = exit_period
        self.atr_period = atr_period
        self.risk_ratio = risk_ratio
        self.max_units = max_units
        self.add_interval = add_interval
        self.stop_loss = stop_loss
        self.contract_multiplier = contract_multiplier
        
        # 账户状态
        self.cash = 0
        self.position = 0  # 持仓数量
        self.units = 0  # 当前 Unit 数
        self.entry_price = 0
        self.last_add_price = 0
        
        # 历史记录
        self.trades: List[Dict] = []
        self.equity_curve: List[float] = []
        
    def calculate_atr(self, df: pd.DataFrame) -> pd.Series:
        """计算 ATR"""
        high = df['high']
        low = df['low']
        close_prev = df['close'].shift(1)
        
        tr1 = high - low
        tr2 = abs(high - close_prev)
        tr3 = abs(low - close_prev)
        
        tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
        atr = tr.ewm(alpha=1/self.atr_period, min_periods=self.atr_period).mean()
        
        return atr
    
    def calculate_unit(self, account_value: float, atr: float) -> int:
        """计算 Unit 仓位"""
        risk_amount = account_value * self.risk_ratio
        unit = int(risk_amount / (atr * self.contract_multiplier))
        return max(unit, 0)
    
    def run(self, df: pd.DataFrame, initial_cash: float = 1000000) -> Dict:
        """
        运行回测
        
        参数：
            df: DataFrame，需包含 high, low, close 列
            initial_cash: 初始资金
        返回：
            回测结果字典
        """
        self.cash = initial_cash
        df = df.copy()
        
        # 计算 ATR
        df['atr'] = self.calculate_atr(df)
        
        # 计算指标
        df['highest_n'] = df['high'].rolling(self.entry_period).max()
        df['lowest_m'] = df['low'].rolling(self.exit_period).min()
        
        # 生成信号
        df['signal'] = 0
        df.loc[df['close'] > df['highest_n'].shift(1), 'signal'] = 1  # 入场
        df.loc[df['close'] <= df['lowest_m'].shift(1), 'signal'] = -1  # 出场
        
        # 回测主循环
        for i in range(self.entry_period, len(df)):
            row = df.iloc[i]
            current_price = row['close']
            atr = row['atr']
            
            # 更新权益曲线
            position_value = self.position * current_price
            total_value = self.cash + position_value
            self.equity_curve.append(total_value)
            
            # ========== 无持仓 ==========
            if self.position == 0:
                # 入场信号
                if row['signal'] == 1:
                    unit_size = self.calculate_unit(total_value, atr)
                    if unit_size > 0:
                        self.position = unit_size
                        self.cash -= unit_size * current_price
                        self.units = 1
                        self.entry_price = current_price
                        self.last_add_price = current_price
                        
                        self.trades.append({
                            'entry_date': row.name,
                            'entry_price': current_price,
                            'type': 'LONG',
                            'units': 1
                        })
            
            # ========== 有持仓 ==========
            else:
                # 检查加仓
                if self.units < self.max_units:
                    add_threshold = self.last_add_price + self.add_interval * atr
                    if current_price >= add_threshold:
                        unit_size = self.calculate_unit(total_value, atr)
                        if unit_size > 0:
                            self.position += unit_size
                            self.cash -= unit_size * current_price
                            self.units += 1
                            self.last_add_price = current_price
                            
                            self.trades[-1]['units'] = self.units
                            self.trades[-1]['last_add_price'] = current_price
                
                # 检查止损
                stop_price = self.entry_price - self.stop_loss * atr
                if current_price <= stop_price:
                    # 止损平仓
                    pnl = (current_price - self.trades[-1]['entry_price']) * self.position
                    self.trades[-1]['exit_date'] = row.name
                    self.trades[-1]['exit_price'] = current_price
                    self.trades[-1]['pnl'] = pnl
                    self.trades[-1]['exit_reason'] = 'STOP_LOSS'
                    
                    self.cash += self.position * current_price
                    self.position = 0
                    self.units = 0
                
                # 检查出场
                elif row['signal'] == -1:
                    pnl = (current_price - self.trades[-1]['entry_price']) * self.position
                    self.trades[-1]['exit_date'] = row.name
                    self.trades[-1]['exit_price'] = current_price
                    self.trades[-1]['pnl'] = pnl
                    self.trades[-1]['exit_reason'] = 'EXIT_SIGNAL'
                    
                    self.cash += self.position * current_price
                    self.position = 0
                    self.units = 0
        
        # 计算结果
        return self._calculate_results()
    
    def _calculate_results(self) -> Dict:
        """计算回测结果"""
        if not self.trades:
            return {
                'total_trades': 0,
                'winning_trades': 0,
                'losing_trades': 0,
                'win_rate': 0,
                'total_pnl': 0,
                'annual_return': 0,
                'max_drawdown': 0,
                'sharpe_ratio': 0
            }
        
        # 过滤已完成交易
        completed_trades = [t for t in self.trades if 'exit_price' in t]
        
        winning = [t for t in completed_trades if t['pnl'] > 0]
        losing = [t for t in completed_trades if t['pnl'] <= 0]
        
        total_pnl = sum(t['pnl'] for t in completed_trades)
        
        # 计算夏普比率
        returns = pd.Series(self.equity_curve).pct_change().dropna()
        sharpe = np.sqrt(252) * returns.mean() / returns.std() if returns.std() > 0 else 0
        
        # 计算最大回撤
        equity = pd.Series(self.equity_curve)
        peak = equity.expanding().max()
        drawdown = (equity - peak) / peak
        max_drawdown = drawdown.min() * 100
        
        return {
            'total_trades': len(completed_trades),
            'winning_trades': len(winning),
            'losing_trades': len(losing),
            'win_rate': len(winning) / len(completed_trades) if completed_trades else 0,
            'total_pnl': total_pnl,
            'avg_win': np.mean([t['pnl'] for t in winning]) if winning else 0,
            'avg_loss': np.mean([t['pnl'] for t in losing]) if losing else 0,
            'max_drawdown': max_drawdown,
            'sharpe_ratio': sharpe,
            'final_value': self.equity_curve[-1] if self.equity_curve else 0,
            'trades': completed_trades
        }
```

## 11. 实盘注意事项

### 11.1 交易前准备

**1. 数据准备**
- 使用**实时数据源**（如券商 API、Wind、同花顺）
- 建议每日收盘后下载前一日数据
- 盘中使用前日数据计算指标

**2. 交易时间**
- A股：9:30-11:30、13:00-15:00
- 建议在 **14:55 之后**执行交易信号（尾盘确认）
- 避免集合竞价时间交易

**3. 交易成本**
| 成本类型 | 费率 |
|---------|------|
| 佣金 | 万 1-3（最低 5 元） |
| 印花税 | 千 1（卖出时）|
| 过户费 | 万 0.1（沪市） |
| 滑点 | 建议 0.05%-0.1% |

### 11.2 入场执行要点

**入场条件确认**：
1. 收盘价 > 20 日最高（信号确认）
2. 检查是否为假信号（前一日已突破但回落）
3. 确认当日成交量正常（避免对倒）

**下单技巧**：
- 使用**市价单**或**涨停价买入**（确保成交）
- 不要使用限价单（可能无法成交）
- 建议分 2-3 批下单（降低冲击成本）

### 11.3 加仓执行要点

**加仓条件确认**：
1. 价格从加仓点继续上涨 ≥ 0.5 ATR
2. 账户仍有足够资金
3. 未触发止损

**加仓风险**：
- 加仓会**增加持仓成本**
- 如果行情反转，**亏损会加速**
- 建议加仓时同步上移止损价

### 11.4 止损执行要点

**止损纪律**：
- **绝对不允许手动取消止损**
- 止损是 **2 ATR**，不是 5%、不是 10%
- 触发止损后 **必须执行**，不能犹豫

**止损后的处理**：
1. 平仓后该品种进入"观察期"
2. 20 个交易日内不再交易该品种
3. 等待下一次明确的突破信号

### 11.5 出场执行要点

**出场信号**：
- 跌破 10 日最低价
- 建议尾盘确认后执行

**出场方式**：
- 使用**市价单**确保成交
- 如果持仓量大，分 2 批卖出

### 11.6 仓位管理要点

**资金分配**：
```
单品种最大风险 = 账户 × 2% × 4 Unit = 账户 × 8%
同时持仓最大风险 = 账户 × 20%（约 4-5 个品种）
```

**避免的操作**：
- ❌ 不要在亏损时加仓（摊平成本）
- ❌ 不要重仓单一品种
- ❌ 不要使用高杠杆
- ❌ 不要在流动性差的品种上重仓

### 11.7 心理建设

**必须做到**：
- ✅ 严格遵守止损纪律
- ✅ 接受连续 3-5 次止损
- ✅ 不因为连续亏损而降低风险
- ✅ 不因为害怕错过而追高

**必须避免**：
- ❌ 手动提前止损
- ❌ 盈利时过早止盈
- ❌ 亏损时死扛
- ❌ 因情绪改变规则

## 12. 常见误区与解决方案

### 误区一：参数过拟合

**表现**：
- 在历史数据上表现极好，实盘很差
- 过度优化参数（如精确到小数点后两位）

**原因**：
- 历史数据有限，过度优化导致泛化能力差
- 市场结构变化，参数失效

**解决方案**：
- 使用 **Walk-Forward** 分析验证参数
- 参数保留 1-2 位小数
- 定期（季度）重新验证参数有效性

### 误区二：忽视交易成本

**表现**：
- 回测收益 20%，实盘只有 10%
- 高频交易时成本占比大

**原因**：
- 滑点估计不足
- 手续费计算不完整
- 冲击成本未考虑

**解决方案**：
- 回测时加入 **0.1%-0.2%** 的滑点
- 手续费按实际费率计算
- 大额交易分批执行

### 误区三：仓位管理不当

**表现**：
- 连续止损后账户大幅回撤
- 某一品种波动大导致亏损超标

**原因**：
- 未按 ATR 计算仓位
- 多个品种同时触发止损

**解决方案**：
- 严格执行 **Unit = 风险金额 / (ATR × 乘数)**
- 品种间仓位分配要均衡
- 总持仓风险不超过账户 20%

### 误区四：心理偏差

**表现**：
- 止损触发但手动取消
- 盈利时过早止盈
- 亏损时死扛

**原因**：
- 人类天然厌恶损失
- 情绪影响判断

**解决方案**：
- **100% 程序化交易**，不带情绪
- 制定交易计划并严格执行
- 每日/每周复盘交易记录

### 误区五：过度交易

**表现**：
- 每天都要交易
- 任何波动都想抓住

**原因**：
- 缺乏耐心
- 过度自信

**解决方案**：
- 制定交易频率目标（如每月 1-2 次）
- 没有信号时**绝对不交易**
- 学会空仓等待

### 误区六：品种选择不当

**表现**：
- 波动太小的品种反复止损
- 流动性差的品种滑点大

**原因**：
- 未做品种筛选
- 追求小概率机会

**解决方案**：
- 只交易 **ATR/价格 > 0.5%** 的品种
- 日均成交额 > 1000 万
- 参考第 8 节的品种筛选标准

### A. ATR 计算完整代码

```python
import pandas as pd
import numpy as np

def calculate_atr(df, period=14):
    """
    完整的 ATR 计算函数
    
    参数：
        df: DataFrame，需包含 high, low, close 列
        period: ATR 周期，默认 14
    返回：
        ATR Series
    """
    high = df['high']
    low = df['low']
    close = df['close']
    
    # 计算真实波幅（TR）
    tr1 = high - low
    tr2 = abs(high - close.shift(1))
    tr3 = abs(low - close.shift(1))
    tr = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
    
    # 计算 ATR（使用指数移动平均）
    atr = tr.ewm(alpha=1/period, min_periods=period).mean()
    
    return atr
```
