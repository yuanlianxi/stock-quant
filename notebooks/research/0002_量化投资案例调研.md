#### 案例一：趋势交易者老张
- **背景：** 2015年开始接触量化，IT工程师出身
- **策略选择：** 双均线趋势跟踪 + 布林带止损

  - 严格执行止损，年度最大回撤控制在 15%

#### 案例二：网格交易者小李
- **背景：** 2019年开始，用网格策略操作 ETF
- **策略选择：** 网格交易（ Bands 策略）
- **收益表现：** 2019-2022年，年化约 12-15%
- **成功经验：**
  - 选择低波动ETF（510050、510300）作为标的
  - 网格参数：间距 3-5%，每格等资金
  - 不止损，越跌越买，耐心等待回归

## 2. 失败案例分析

### 2.1 机构失败案例

#### 长期资本管理公司（LTCM）
- **时间：** 1994-2000年
- **背景：** 诺贝尔奖得主默顿、米勒等人创立，被誉为"梦之队"
- **策略：** 固定收益套利（收敛交易）
- **失败原因：**
  - **模型风险：** 历史数据样本外失效，俄罗斯国债违约触发"流动性陷阱"
  - **杠杆过高：** 杠杆率高达 25-30 倍
  - **尾部风险：** 小概率事件（"黑天鹅"）处理不足
  - 1998年 4 个月内亏损 45 亿美元，最终被接管
- **教训总结：**
  - 模型不可能预测所有极端事件
  - 杠杆是把双刃剑，放大收益也放大风险
  - 流动性风险管理是生命线

#### 百度风投（Baidu Ventures）量化项目
- **失败原因：** 
  - 策略同质化严重，在市场风格切换时集体失效
  - 高频策略在竞争激烈的A股市场手续费侵蚀利润
- **教训：** 量化不等于赚钱，差异化策略和成本控制同样重要

#### 2021-2022年国内量化私募集体回撤
- **背景：** 2021年9月后，多家百亿量化私募产品大幅回撤
- **原因：**
  - 抱团小市值股票策略失效（2021年11月后小盘股暴跌）
  - 策略同质化严重，踩踏出逃
  - 部分产品杠杆过高
- **教训：** 策略要适应市场变化，没有永远的圣杯

## 3. 实操策略详解

### 3.1 趋势跟踪策略 — 双均线策略

#### 原理
短期均线上穿长期均线（金叉）买入，下穿（死叉）卖出。核心假设：趋势会延续。

#### 指标参数
```
短期均线：MA5 或 MA10
长期均线：MA20 或 MA60
```

#### 实操步骤
1. 每日收盘后计算 MA5、MA20
2. 若 MA5 上穿 MA20 → 次日开盘买入
3. 若 MA5 下穿 MA20 → 次日开盘卖出
4. 仓位管理：每次买入固定仓位（如 10%），不满仓

#### 优缺点
- **优点：** 简单易执行，符合趋势时收益可观
- **缺点：** 震荡行情反复止损，趋势假信号多

#### 改进方案
- 增加过滤条件：MACD>0 或成交量放大才入场
- 引入布林带确认：价格站上布林中轨才做多

---

### 3.2 布林带策略

#### 原理
利用统计学原理，价格围绕均线（MA20）上下波动 2 个标准差。价格触及上轨超买，触及下轨超卖。

#### 指标参数
```
中轨：MA20
上轨：MA20 + 2σ
下轨：MA20 - 2σ
```

#### 实操步骤
1. 价格触及下轨且缩量 → 买入
2. 价格触及上轨且放量 → 卖出
3. 止损设置：跌破下轨 2% 止损

#### 优缺点
- **优点：** 有明确边界，适合震荡市
- **缺点：** 趋势行情中会长期持有亏损仓位

---

### 3.3 MACD策略

#### 原理
MACD 由快线（DIF）、慢线（DEA）和柱状图（MACD柱）组成。用于判断趋势方向和动量。

#### 指标参数（标准参数）
```
DIF快线：EMA12 - EMA26
DEA慢线：DIF的EMA9
MACD柱：DIF - DEA
```

#### 实操步骤
1. DIF 上穿 DEA（金叉）+ MACD柱由负转正 → 买入
2. DIF 下穿 DEA（死叉）+ MACD柱由正转负 → 卖出
3. 配合 0 轴使用：DIF 和 DEA 都在 0 轴上方是多头市场

#### 优缺点
- **优点：** 信号可靠，参数经典
- **缺点：** 滞后性强，趋势初期反应慢

---

### 3.4 多因子选股策略

#### 原理
通过多个因子（价值、动量、质量、规模等）对股票打分，选取高分股票构建组合。

#### 常用因子
| 因子类别 | 因子名 | 说明 |
|---------|--------|------|
| 价值 | PE、PB、PS | 估值越低越好 |
| 动量 | N日收益率 | 追涨杀跌 |
| 质量 | ROE、毛利率 | 盈利能力 |
| 规模 | 市值 | 小盘/大盘轮动 |
| 情绪 | 换手率变化 | 资金流入 |

#### 实操步骤
1. 选取候选池（如中证500成分股）
2. 每个因子标准化（Z-score）
3. 加权求和得到总分
4. 选取前 10% 或前 20 只
5. 等权配置，每季度调仓

#### 优缺点
- **优点：** 策略逻辑清晰，可解释性强
- **缺点：** 因子会失效，需要持续迭代

---

### 3.5 网格交易策略

#### 原理
在固定价格区间内设置网格，每格等间距挂单。价格下跌买入，上涨卖出，自动化高抛低吸。

#### 指标参数
```
标的选择：低波动ETF（510050、510300）
网格间距：3%-5%
每格仓位：等资金（如每次买1手）
总仓位上限：不超过可用资金的60%
```

#### 实操步骤
1. 确定网格区间（如 2.8-3.5 元）
2. 计算网格数量：区间 / 间距 = 网格数
3. 在每个网格挂买单和卖单
4. 成交后自动补齐对应卖单/买单
5. 区间外不操作，等待下次机会

#### 优缺点
- **优点：** 自动化程度高，心理压力小，适合震荡市
- **缺点：** 单边趋势行情会卖飞或套牢，需要预留资金

### 3.6 海龟交易策略

#### 原理
经典的趋势跟踪系统，1983年由理查德·丹尼斯创立。核心是"趋势终将延续"，用ATR（平均真实波幅）管理仓位。

#### 指标参数
```
入场信号：价格突破20日最高价（做多）或20日最低价（做空）
出场信号：价格跌破10日最低价（做多）或10日最高价（做空）
仓位计算：账户风险 / (ATR × 每手价值)
止损：入场价 - 2ATR
```

#### 实操步骤
1. 计算标的 ATR（推荐周期14）
2. 确定单次交易允许亏损（账户的1-2%）
3. 计算仓位数量
4. 价格突破20日高点 → 买入 1 Unit
5. 每上涨 0.5ATR 加仓 1 Unit（最多4个Unit）
6. 价格跌破10日低点 → 全平

#### 优缺点
- **优点：** 系统化，规则明确，有完整的资金管理
- **缺点：** 需要足够耐心，震荡市频繁止损

#### 海龟交易法则速查表
| 参数 | 值 |
|------|-----|
| 入场周期 | 20日 |
| 出场周期 | 10日 |
| 加仓间隔 | 0.5ATR |
| 最大Unit数 | 4 |
| 单次最大亏损 | 账户 2% |

## 4. 风险提示

1. **模型风险：** 历史不代表未来，策略会失效
2. **市场风险：** A股散户占比高，风格切换快，策略需快速迭代
3. **流动性风险：** 小盘股策略在极端行情可能无法出货
4. **杠杆风险：** 融资融券放大波动，慎用杠杆
5. **操作风险：** 程序故障、网络中断可能导致意外交易
6. **心理风险：** 执行纪律比策略本身更难
7. **政策风险：** 监管政策变化可能影响策略有效性（如涨停板制度、T+0预期）

## 5. 建议

### 入门路径
1. **先模拟盘** 验证策略至少 6 个月，不急于实盘
2. **从小资金开始** 第一年资金不超过可投资资产的 20%
3. **策略优先简单** 复杂≠有效，简单策略更容易坚持
4. **建立日志** 记录每笔交易的原因和结果，定期复盘

### 策略选择建议
| 市场环境 | 推荐策略 | 回避策略 |
|---------|---------|---------|
| 趋势明确 | 双均线、海龟 | 网格、均值回归 |
| 震荡整理 | 布林带、MACD背离 | 趋势跟踪 |
| 高波动 | 轻仓+宽止损 | 重仓+窄止损 |
| 低波动 | 网格交易 | 趋势策略 |
