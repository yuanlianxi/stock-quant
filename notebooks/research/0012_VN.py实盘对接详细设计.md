## 3. 转译方案

### 3.1 策略转译（推荐）

将 Backtrader 策略改写为 VN.py CTA 策略格式。

| Backtrader | VN.py |
|-----------|-------|
| `bt.Strategy` | `CtaStrategy` |
| `bt.indicators` | `talib` 或 `numpy` |
| `self.buy()` / `self.sell()` | `self.buy()` / `self.sell()` |
| `self.data` | `self.am`（K线数据）|
| `next()` | `on_bar()` |

### 3.2 转译示例（双均线策略）

**Backtrader 版本**：
```python
class DualMA(bt.Strategy):
    params = (('fast', 5), ('slow', 20))
    
    def __init__(self):
        self.ma_fast = bt.indicators.SMA(self.data, period=self.p.fast)
        self.ma_slow = bt.indicators.SMA(self.data, period=self.p.slow)
    
    def next(self):
        if self.ma_fast > self.ma_slow and not self.position:
            self.buy()
        elif self.ma_fast < self.ma_slow and self.position:
            self.sell()
```

**VN.py 版本**：
```python
class DualMAStrategy(CtaStrategy):
    fast_ma = 5
    slow_ma = 20
    parameters = ['fast_ma', 'slow_ma']
    
    def __init__(self):
        self.fast_ma = ta.MA(self.am, self.fast_ma)
        self.slow_ma = ta.MA(self.am, self.slow_ma)
    
    def on_bar(self, bar: BarData):
        if self.pos:
            return
        
        if self.fast_ma > self.slow_ma:
            self.buy(bar.close, 1)
        elif self.fast_ma < self.slow_ma:
            self.sell(bar.close, 1)
```
